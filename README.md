# 🎵 音声録音・再生アプリ

Node.jsで作成されたコマンドライン音声録音・保存・再生アプリです。

## 🚀 機能

- 🎤 **音声録音**: マイクから音声を録音（.wav形式）
- 💾 **ファイル保存**: 録音した音声を自動保存
- 🔊 **音声再生**: 保存した音声ファイルを再生
- 📝 **ファイル管理**: 録音ファイル一覧表示・選択再生

## 📋 必要要件

### システム要件
- Node.js (v14以上推奨)
- Windows 10/11 (このアプリはWindows環境でテスト済み)

### 外部ツール（Windows環境）
Windows環境で録音機能を使用するには、**SoX** をインストールする必要があります：

1. [SoX公式サイト](http://sox.sourceforge.net/) からダウンロード
2. または Chocolatey を使用：
   ```bash
   choco install sox
   ```
3. または Scoop を使用：
   ```bash
   scoop install sox
   ```

## 🛠 インストール

1. プロジェクトをクローンまたはダウンロード
2. 依存関係をインストール：
   ```bash
   npm install
   ```

## 🎮 使用方法

### アプリケーション起動
```bash
npm start
```
または
```bash
node index.js
```

### メニュー操作

アプリケーション起動後、以下のメニューが表示されます：

```
🎵 音声録音・再生アプリ
========================
1. 録音する (5秒)
2. 録音する (カスタム時間)
3. 録音ファイル一覧
4. 録音を再生
5. 終了
========================
```

#### 1. 録音する (5秒)
- 5秒間の録音を開始
- 録音中は `Ctrl+C` で手動停止可能
- 録音ファイルは `recordings/` フォルダに自動保存

#### 2. 録音する (カスタム時間)
- 任意の時間（秒単位）で録音
- 例：`10` と入力すると10秒間録音

#### 3. 録音ファイル一覧
- 保存された録音ファイルの一覧を表示
- ファイル名、サイズ、作成日時を表示

#### 4. 録音を再生
- 録音ファイル一覧から再生したいファイルを選択
- 番号を入力して再生

#### 5. 終了
- アプリケーションを終了

## 🔧 技術仕様

- **録音形式**: WAV (16kHz, 16bit)
- **録音ライブラリ**: node-record-lpcm16
- **再生ライブラリ**: play-sound
- **ファイルシステム**: Node.js fs モジュール

## ⚠️ 注意事項

1. **マイクアクセス**: 初回実行時にマイクへのアクセス許可が必要です
2. **録音品質**: 録音品質はマイクの性能に依存します
3. **ファイルサイズ**: 長時間録音は大きなファイルサイズになります
4. **Windows環境**: SoXのインストールが必要です

## 🐛 トラブルシューティング

### 録音ができない場合
1. SoXがインストールされているか確認
2. マイクが正しく接続されているか確認
3. マイクのアクセス許可を確認

### 再生ができない場合
1. 音声出力デバイスが正しく設定されているか確認
2. ファイルが破損していないか確認
